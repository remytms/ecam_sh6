Compile
-------
### With Waf
Run:

    $ python waf configure
    $ python waf build
    $ python waf build

The build must be run two times.

### With custom script
Run:

    $ ./compile

Run sh6
-------
Run in interactive shell mode:

    $ ./sh6

Run in batch mode:

    $ ./sh6 my_batch_file

Notes
----
Waf version: 1.9.6
